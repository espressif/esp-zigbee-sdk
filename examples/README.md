# Examples

This directory contains a range of ESP Zigbee examples. These examples are intended to demonstrate different functionalities of usage Zigbee protocol to build Zigbee product, and to provide code that you can copy and adapt into your own projects.  

## Examples Layout
The examples are grouped into subdirectories by category. Each category directory contains one or more example projects:
- [all_device_types_app](all_device_types_app) Zigbee all device types example in command line interface mode
- [customized_devices](customized_devices) Zigbee fully customized attribute, cluster, endpoint and functions for server and client devices
- [zigbee_gateway](zigbee_gateway) Zigbee gateway example, runs on a Wi-Fi SoC such as ESP32, ESP32-C3 ESP32-C6 and ESP32-S3, with an 802.15.4 SoC like ESP32-H2 running [ot_rcp example](https://github.com/espressif/esp-idf/tree/master/examples/openthread/ot_rcp)
- [home_automation_devices](home_automation_devices) Zigbee Home Automation examples including on_off_light, on_off_switch, color_dimmable_light, color_dimmer_switch, temperature sensor, thermostat, etc
- [ota_upgrade](ota_upgrade) Zigbee OTA (Over-The-Air) upgrade examples
- [sleepy_devices](sleepy_devices) Zigbee sleepy (light/deep) end device examples
- [touchlink](touchlink) Zigbee TouchLink (initiator/target) examples

## Using Examples
Please see the information on usage under each Zigbee example folder.

Each example is a standalone project. You can copy an example directory to anywhere that you can modify and work with.
